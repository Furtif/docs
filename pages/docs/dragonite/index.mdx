# Dragonite

[**Dragonite**](https://github.com/UnownHash/Dragonite-Public) is our backend controller. This service is used to connect and manage multiple workers. In short, it tells the game which accounts to log in with, where to go, and what action to take.

## Dragonite-Admin

[**Dragonite-Admin**](https://github.com/UnownHash/Dragonite-Public) is a web front end to interact with Dragonite. Generally speaking this is a proxy service to allow Admin users to interact with the service. With this service you can configure your Areas, Accounts, Routes and schedules.
