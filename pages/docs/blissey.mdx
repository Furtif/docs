# Blissey

[**Blissey**](https://github.com/UnownHash/Blissey) processes reporting data from Golbat into separate database tables for visualization in [Grafana](https://grafana.com/).

## Terminology

* **rpl**: reporting length (period), stored for intervals 5, 15, 60, 1440 and 10080 minutes
* **numWiEnc**: number of pokemon that initial scanned as wild, lateron encountered
* **secWiEnc**: sum of seconds between seen wild and encountered
